# Documentación del Proyecto

Esta carpeta contiene toda la documentación del proyecto MVP - Sistema de Registro de Tareas.

## Estructura

### Documentación Principal

- **`producto.md`** - Descripción del producto, público objetivo y características
- **`arquitectura.md`** - Arquitectura del sistema (Frontend, Backend, DB)
- **`modelo-datos.md`** - Modelo de datos con entidades, relaciones y restricciones
- **`historias-y-tickets.md`** - 55 historias de usuario y 33 tickets técnicos
- **`testing.md`** - Estrategia de testing del MVP
- **`reglas-negocio.md`** - Reglas de negocio consolidadas
- **`consignas-mvp.md`** - Consignas del MVP (resumen)
- **`consignas-mvp-cursor.md`** - Consignas consolidadas con estado de artefactos

### Carpetas Especializadas

- **`api/`** - Contratos y especificaciones de API
- **`backend/`** - Documentación específica del backend (Laravel)
- **`frontend/`** - Documentación específica del frontend (React)
- **`domain/`** - Modelo de datos técnico detallado
- **`ai/`** - Guías de uso de IA
- **`plantillas/`** - Plantillas para documentación

### Carpetas Temporales

- **`_ENTREGA-1/`** - Archivos de verificación de la Entrega 1
