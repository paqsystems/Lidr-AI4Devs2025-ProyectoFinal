# Producto

## Visión del Producto

### Descripción General

Proyecto final del Master de Lidr AI4Devs 2025 que integra herramientas de desarrollo, gestión de proyectos y bases de datos mediante el protocolo MCP (Model Context Protocol).

### Propósito

El producto busca facilitar la gestión y automatización de tareas de desarrollo mediante la integración de múltiples servicios y herramientas en un ecosistema unificado.

## Características Principales

### Integraciones

- **Jira**: Gestión de issues y tickets
- **Bases de Datos**: Conexión con MySQL y MSSQL
- **Playwright**: Automatización de pruebas y navegación web
- **MCP Toolbox**: Herramientas de base de datos

### Funcionalidades

- Listado y gestión de issues de Jira
- Consultas y operaciones sobre bases de datos
- Automatización de tareas mediante agentes
- Interfaz unificada para múltiples servicios

## Público Objetivo

- Desarrolladores que trabajan con múltiples herramientas
- Equipos que necesitan automatizar tareas repetitivas
- Proyectos que requieren integración entre sistemas

## Roadmap

### Versión 1.0 (MVP)

- Integración básica con Jira

### Versión 2.0

- Agentes inteligentes para automatización
