## Notas de Implementación

1. **Orden de Ejecución**: Ejecuta primero los CREATE TABLE, luego los ALTER TABLE para agregar Foreign Keys.

2. **Foreign Keys Comentadas**: Las Foreign Keys están comentadas porque algunas referencian tablas que no están en esta lista. Descomenta solo las que necesites y asegúrate de que las tablas referenciadas existan.

3. **Valores por Defecto**: Algunos DEFAULT values pueden necesitar ajuste según tu versión de SQL Server (especialmente fechas y valores numéricos).

4. **ROW_VERSION**: Las columnas `ROW_VERSION` de tipo `TIMESTAMP` son generadas automáticamente por SQL Server y no deben especificarse en INSERT.

5. **Índices**: Los índices no clúster no están incluidos en estos scripts. Si los necesitas, puedes generarlos por separado.
